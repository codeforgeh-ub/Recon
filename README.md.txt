# OSINT Recon Project

A structured open-source intelligence (OSINT) project focused on analyzing and documenting publicly accessible data related to a target organization. This repository contains detailed reconnaissance outputs including domain mapping, email enumeration, metadata analysis, and tooling documentation.

---

## 🔍 Project Scope

This repository includes:
- Subdomain enumeration and infrastructure mapping
- Leaked email and social profile discovery
- Metadata extraction from public documents
- Notes on tools and methodologies used
